## 1. ゲームドメイン用語

### ゲーム進行

| 用語 | 読み | 定義 | コード上の名前 |
|------|------|------|-------------|
| ゲーム | げーむ | 1回のプレイセッション全体 | `game` |
| ターン | たーん | 1人のプレイヤーが行動する単位 | `turn` |
| フェーズ | ふぇーず | ターン内の処理段階 | `phase` |
| 年数 | ねんすう | ゲームの総プレイ期間 | `totalYears` |
| 現在年 | げんざいねん | 現在何年目か | `currentYear` |
| 現在月 | げんざいつき | 現在何月か（1〜12） | `currentMonth` |
| 年末決算 | ねんまつけっさん | 年末に物件収益を精算するイベント | `yearEnd` |

### プレイヤー

| 用語 | 読み | 定義 | コード上の名前 |
|------|------|------|-------------|
| プレイヤー | ぷれいやー | ゲームに参加する操作主体 | `player` |
| 人間プレイヤー | にんげんぷれいやー | ユーザーが操作するプレイヤー | `PlayerType: 'human'` |
| CPU | しーぴーゆー | コンピューターが操作するプレイヤー | `PlayerType: 'cpu'` |
| 難易度 | なんいど | CPUの行動レベル | `difficulty` |
| 所持金 | しょじきん | プレイヤーが現在持っているお金 | `money` |
| 総資産 | そうしさん | 所持金 + 保有物件の合計評価額 | `totalAssets` |
| コマ | こま | マップ上でプレイヤーを示す駒 | `pawn` |

### マップ・移動

| 用語 | 読み | 定義 | コード上の名前 |
|------|------|------|-------------|
| マップ | まっぷ | 日本全国の路線ボード全体 | `board` |
| 都市 | とし | 物件を購入できる拠点 | `city` |
| マス目 | ますめ | プレイヤーが止まれるボード上の1マス | `square` |
| 路線 | ろせん | 都市間を繋ぐ移動経路 | `route` |
| 新幹線 | しんかんせん | 主要都市間を結ぶ高速路線 | `RouteType: 'shinkansen'` |
| 在来線 | ざいらいせん | 一般的な鉄道路線 | `RouteType: 'local'` |
| フェリー | ふぇりー | 海を渡る船の路線 | `RouteType: 'ferry'` |
| 分岐点 | ぶんきてん | 複数の進路に分かれるマス | `junction` |
| 位置 | いち | プレイヤーが現在いる都市とマス目の組 | `position` |

### 物件

| 用語 | 読み | 定義 | コード上の名前 |
|------|------|------|-------------|
| 物件 | ぶっけん | 都市内で購入できる資産 | `property` |
| 物件価格 | ぶっけんかかく | 物件の購入に必要な金額 | `price` |
| 収益 | しゅうえき | 物件から毎年得られる収入 | `income` |
| アップグレード | あっぷぐれーど | 物件に追加投資して収益を増やすこと | `upgrade` |
| 独占 | どくせん | 1都市の全物件を1人が保有している状態 | `monopoly` |
| 独占ボーナス | どくせんぼーなす | 独占時に収益が倍増するボーナス | `monopolyBonus` |
| 地代 | じだい | 他プレイヤーの物件マスに止まった時に支払うお金 | `landFee` |

### カード

| 用語 | 読み | 定義 | コード上の名前 |
|------|------|------|-------------|
| カード | かーど | プレイヤーが使用できるアイテム | `card` |
| 手札 | てふだ | プレイヤーが現在持っているカードの一覧 | `hand` |
| カードマス | かーどます | 止まるとカードを入手できるマス | `SquareType: 'card'` |
| カード売り場 | かーどうりば | カードを購入できるマス | `SquareType: 'shop'` |
| 移動カード | いどうかーど | 移動に関わる効果を持つカード | `CardType: 'move'` |
| 妨害カード | ぼうがいかーど | 他プレイヤーを妨害する効果のカード | `CardType: 'obstruct'` |
| 援助カード | えんじょかーど | 自分を有利にする効果のカード | `CardType: 'assist'` |
| ボンビーカード | ぼんびーかーど | ボンビーに関わる効果のカード | `CardType: 'bombee'` |

### 目的地・ボンビー

| 用語 | 読み | 定義 | コード上の名前 |
|------|------|------|-------------|
| 目的地 | もくてきち | そのターンの到着ボーナスが得られる都市 | `destination` |
| 目的地ボーナス | もくてきちぼーなす | 目的地に先着した時に得られる報酬 | `destinationBonus` |
| ボンビー | ぼんびー | 最下位プレイヤーに憑いて妨害する貧乏神 | `bombee` |
| ミニボンビー | みにぼんびー | ボンビーの最弱形態 | `BombeeType: 'mini'` |
| キングボンビー | きんぐぼんびー | ボンビーの最強形態 | `BombeeType: 'king'` |
| 憑依 | ひょうい | ボンビーがプレイヤーに取り憑くこと | `attach` |

## 3. 技術用語

| 用語 | 定義 | 使用箇所 |
|------|------|---------|
| GameState | ゲーム全体の状態を表すオブジェクト | `src/types/game.ts` |
| GameManager | ゲーム進行全体を管理するクラス | `src/game/GameManager.ts` |
| SaveManager | セーブ・ロード処理を抽象化するインターフェース | `src/save/ISaveManager.ts` |
| Scene | Phaser.jsの画面単位 | `src/scenes/` |
| Tween | Phaser.jsのアニメーション機能 | シーン内のアニメーション処理 |
| Registry | Phaser.jsのシーン間データ共有機能 | シーン間の状態受け渡し |
